# 2026-05-20 Pro CLI v2

## Context

Implemented the `plans/20260520-1700-typeburn-pro-cli-v2/` big-bang CLI plan.

## What Changed

- Added `internal/runner` shared session construction.
- Added fang/cobra CLI commands: `run`, `history`, `version`, `config`, `replay`.
- Added raw `run --no-tui` runner with terminal restore lifecycle tests.
- Added JSON/table renderers, replay schema fixture, CI size/no-exit gates.
- Updated README, CLI reference, changelog, roadmap, architecture, and contribution rules.

## Decisions

- Fang `v1.0.0`, cobra `v1.10.2`, and `golang.org/x/term v0.43.0` are pinned.
- `config set` rejects invalid values before write.
- Root unknown args keep v1 TUI fall-through; subcommands are strict.
- Size cap locked at 8 MiB after measured build size of 5,302,642 bytes.
